# Checkpoints

Download the following checkpoints and place them in this directory:

| File | Description | Size | Link |
|------|-------------|------|------|
| `mtloc_445k.ckpt` | MTLoc best checkpoint (FPN adapter, 445K steps) | ~193 MB | [Google Drive](https://drive.google.com/drive/folders/1UhLLG3vqSiYgx61SUFOjpuUqf1THN1mc?usp=drive_link) |
| `orienternet_mgl.ckpt` | OrienterNet pretrained on MGL (required) | ~630 MB | [Google Drive](https://drive.google.com/drive/folders/1UhLLG3vqSiYgx61SUFOjpuUqf1THN1mc?usp=drive_link) |
| `epoch-195.pth` | YOLOPX pretrained weights (required) | ~379 MB | [Google Drive](https://drive.google.com/drive/folders/1UhLLG3vqSiYgx61SUFOjpuUqf1THN1mc?usp=drive_link) |

After downloading, your directory should look like:
```
checkpoints/
├── mtloc_445k.ckpt
├── orienternet_mgl.ckpt
└── epoch-195.pth
```
